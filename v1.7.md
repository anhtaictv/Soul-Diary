# Kế hoạch nâng cấp v1.7 — Soul Diary

## Ý tưởng đã thảo luận

### ✅ 1. Admin Tiếp Cận Người Dùng (Hộp thư) — *Đang triển khai*
Admin/counselor chủ động gửi tin đến user có tâm trạng tệ.  
**Loại tin:** 💬 Tin nhắn cá nhân / ✨ Động viên nhanh / 🎵 Gợi ý nhạc / 📖 Gợi ý bài viết  
User nhận push notification + đọc trong trang Hộp thư. Badge đỏ trên nav khi có tin chưa đọc.  
Admin thấy last_mood từng user trong bảng, nút 💌 để liên hệ ngay.

---

### 2. Heatmap Cảm xúc Năm (GitHub-style)
Lưới 365 ô, mỗi ô = 1 ngày, tô màu theo avg mood (xanh → đỏ).  
Click vào ô xem entry ngày đó. Frontend-only từ data diary đã có — không cần API mới.  
Đặt trong trang Biểu đồ, toggle cùng với mood calendar.

### 3. Thử thách Sức khỏe Tâm thần 7/14/21 ngày
Gói challenge có cấu trúc: "7 ngày biết ơn", "14 ngày viết CBT", "21 ngày mindful".  
Mỗi ngày có nhiệm vụ nhỏ. Streak riêng cho challenge, tặng huy hiệu khi hoàn thành.  
Bảng `Challenges` + `ChallengeProgress`. Feature flag riêng.

### 4. Xuất Nhật ký (PDF / CSV)
User export toàn bộ hoặc theo khoảng thời gian ra PDF đẹp hoặc CSV.  
Backend: `pdfkit`. Hữu ích để chia sẻ với tâm lý học đường / lưu trữ cá nhân.

### 5. Báo cáo Tháng cho Admin
Dashboard admin thêm biểu đồ xu hướng mood toàn hệ thống theo tuần/tháng.  
Top cảm xúc phổ biến, tỷ lệ user active, số user cải thiện vs xấu đi.  
Giúp nhà trường/counselor nắm bức tranh tổng thể để can thiệp kịp thời.

### 6. Nhắc nhở Tùy chỉnh
User tự set giờ cố định cho push notification thay vì hệ thống tự tính từ lịch sử.  
UI đơn giản: chọn giờ + ngày trong tuần. Lưu vào bảng `Settings` per-user.

### 7. Tâm sự Ẩn danh (Community Wall)
Board nhỏ: học sinh post cảm xúc ẩn danh, người khác bấm "💙 Đồng cảm" (không comment tự do).  
Admin moderation: ẩn/xóa post. Tạo cảm giác không cô đơn mà không có toxic comments.  
**Rủi ro:** cần moderation cẩn thận. Ship sau khi có đủ user base.
